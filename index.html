crypt4gh WebAssembly demo


This application encrypts data using crypt4gh.

A private key (the sender's key) is randomly generated each time the page is loaded, and a public key (the recipient) is hardcoded.

All work is done in your web browser. Your data never leaves your computer.

Start by giving this application temporary permission to read and edit from a directory on your computer. This directory should contain the sensitive data that you want to encrypt.

After you close this tab, the application won't be able to access files on your computer, and you'll need to grant permission again.

Then encrypt it! This dumb demo creates an encrypted copy of every file in the directory
